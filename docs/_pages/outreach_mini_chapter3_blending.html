Measuring Flux

Blending results in light from the neighbors contaminating the flux of the lensed star, fλ,s, measured at time t.

For an isolated lensed star, the measured flux fλ(t) is given by:

$$f_{\lambda}(t) = f_{\lambda,s}(t)A(u(t))$$
In practice what we actually measure is:

$$f_{\lambda}(t) = f_{\lambda, s}(t)A(u(t)) + f_{\lambda, b}(t)$$

where A(u(t)) is the magnification at angular separation u, and fb(t) is the additional flux from all overlapping neighbors.

This is strongly time and wavelength dependent.

Blending in microlensing light curves

Blending can make tE appear shorter, leading to a mis-estimation of the lens parameters.

It can be a symmetrical change in the light curve, mimicing the parallax component πE,⊥.

Notation

Different authors sometimes use different formulations for blending, so you may also see the blend ratio, g:
$$g = f_{b}/f_{s}$$

Dealing with Blending

We need observations at a range of different magnifications


Event OGLE-2024-BLG-0034 observed from (left) South Africa and (right) Australia on 2024/10/12

By measuring the total flux as a function of time, we can find the best-fit model for A(u(t)) that includes both source and blend fluxes.

fs and fb will vary independently as functions of time for each dataset, k (with Nk datapoints).

$$f_{k}(t) = f_{s,k}(t)A(u(t)) + f_{b,k}$$

Measuring the blend flux

A(u(t)) can usually* be considered to be the same for all datasets from Earth.

  1. Estimate initial model parameters
  2. Compute A(u(t)), and model light curve fluxes, fm,k(t)
  3. Evaluate goodness of model fit: $$\chi^{2} = \frac{\sum_{i} \left( \frac{f_{m,k}(t) - f_{k}(t)}{\sigma_{k}(t)} \right)^{2}}{N_{k} - N_{par}},$$ where Npar is the number of fitted parameters, and σk represents the uncertainties of the flux values.
* We'll come back to this later

Measuring the blend flux

Differentiating this expression with respect to each variable in turn and equating it to zero, we can derive a set of simultaneous equations which can be used to infer (fs, fb) for each dataset, for the given magnification model.

Using this set of initial blending parameters, we can now repeat this process, but this time fit for the physical parameters of the amplification model.

Iterating this process, we infer a consistent set of parameters.

Stellar variability

If one or more neighboring stars has intrinsic variability, fb may also vary with time and wavelength.

This can be accommodated by substituting fb(t) with an appropriate function, for instance a linear gradient or sinusoid.

Observations taken before or after the lensing event can be used to determine a suitable model.

Widely-spaced Observatories

We can only assume that the lensing magnification, A(u(t)), is the same for all observers if they share the same line of sight to the event.

If observers are widely separated (e.g. on Earth and a spacecraft), then the observers will measure different angular separations u at time t.

If this is the case, then we have to take this into account when computing A(u(t)).
